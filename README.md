# Student Result Analysis (Python Project)

This project analyzes student data stored in a CSV file using Python.

## Features
- Filter students with marks > 85
- Calculate average marks
- Count students by city
- Find topper
- Filter Jaipur students with marks > 80

## Technologies Used
- Python
- CSV Module

## Project Files
- students.csv
- Student_Result_Analysis (CSV Based).ipynb

## Output
- Displays filtered students
- Calculates average marks
- Shows topper and city-wise distribution
